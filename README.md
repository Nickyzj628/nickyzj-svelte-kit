### 简介

我的个人网站使用 [SvelteKit](https://svelte.dev/docs/kit/introduction) + [UnoCSS](https://unocss.dev/) + [MDUI](https://www.mdui.org/zh-cn/) 搭建。

整合了[前端页面](https://github.com/Nickyzj628/nickyzj-preact)和[后端接口](https://github.com/Nickyzj628/nickyzj-backend)。

### 安装依赖

`pnpm install`

### 运行（开发环境）

`pnpm dev`

### 打包

`pnpm build`

### 运行（生产环境）

`node build`
